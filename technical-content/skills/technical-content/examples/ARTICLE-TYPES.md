# Article Type Classification

## Tutorial
**Goal:** Accomplish task
**Structure:** Prerequisites → Steps with code → Verification

## Deep-Dive
**Goal:** Understand why
**Structure:** Concept → Why it matters → How it works → Applications

## Problem/Solution
**Goal:** Fix error
**Structure:** THE SOLUTION first → Why it works → Alternatives

## Opinion/Strategy
**Goal:** Gain perspective
**Structure:** Position → Context → Evidence → Counterarguments
**Note:** Establish credibility in opening

## Comparison
**Goal:** Make decision
**Structure:** Criteria → Table → Analysis → Recommendations
**Note:** Declare biases upfront
