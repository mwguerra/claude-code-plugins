# Voice Profile Management

Manage your voice profile for consistent writing.

## Arguments

- `setup`: Create new profile (interactive)
- `show`: Display current profile

## Examples

```
/technical-content:voice setup
/technical-content:voice show
```

## Profile Location

Stored at: `docs/voice_profile.md`

Automatically loaded when creating articles.
